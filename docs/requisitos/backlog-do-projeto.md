# Product Backlog

Esse será o backlog utilizado durante o desenvolvimento do sistema, com a sua respectiva priorização utilizando a metodologia **MoSCoW**.

|**Epico**|**Feature**|**User Story**|**Prioridade**|
|-----|-------|----------|----------|
|EP01 - Usuário|FE01 - Autenticação|US01 - Eu, como aluno, gostaria de me cadastrar na plataforma para que eu possa ter acesso às funcionalidades da plataforma.|Must|
|||US02 - Eu, como professor, gostaria de me cadastrar na plataforma para que eu possa ter acesso às funcionalidades da plataforma.|Must|
|||US03 - Eu, como usuário, gostaria de poder realizar o login na plataforma para ter acesso às funcionalidades da plataforma.|Must|
||FE02 - Perfil|US04 - Eu, como usuário, gostaria de ter acesso ao meu perfil para que eu possa ver minhas informações pessoais.|Must|
|||US05 - Eu, como usuário, gostaria de poder editar o meu perfil para manter minhas informações pessoais atualizadas.|Must|
|EP02 - Aulas|FE03 - Visualização|US06 - Eu, como aluno, gostaria de ver informações de um professor para decidir sobre a solicitação da aula.|Should|
||FE04 - Solicitações|US07 - Eu, como aluno, gostaria de poder solicitar aulas para professores disponíveis na plataforma para que eu possa ter aulas particulares online.|Should|
|||US08 - Eu, como professor, gostaria de poder ver todas as solicitações enviadas por alunos para mim, para que eu possa decidir se aceito ou não.|Should|
|||US09 - Eu, como professor, gostaria de poder ver as informações das aulas solicitadas para mim, para que eu possa me decidir se aceito ou não.|Should|
|||US10 - Eu, como professor, gostaria de poder aceitar ou recusar às aulas solicitadas à mim.|Should|
||FE05 - Pedidos|US11 - Eu, como aluno, gostaria de ver todos os pedidos de aulas feitos por mim para professores e o seu status de aceitação para que eu possa pagar.|Should|
|||US12 - Eu, como aluno, gostaria de pagar pelos pedidos de aulas que eu solicitei e que foram aceitos pelos professores.|Could|
|||US13 - Eu, como aluno, gostaria de poder cancelar algum pedido de aula ainda não aceito, caso eu não possa participar no horário que foi marcado por mim.|Could|
|||US14 - Eu, como aluno, gostaria de poder ver o pedido de aula, para ter certeza sobre as informações que eu solicitei.|Should|
||FE06 - Comunicação|US15 - Eu, como professor, gostaria de poder cancelar aulas que foram marcadas pelos alunos, devido à algum justificativa.|Could|
|||US16 - Eu, como aluno, gostaria de poder remarcar aulas que foram canceladas pelo professor para que eu possa ainda ter minha aula.|Could|
|||US17 - Eu, como usuário, gostaria de poder visualizar informações da aula contratada.|Should|
|||US18 - Eu, como usuário, gostaria de enviar mensagens e comentários na página da aula para que eu possa me comunicar com outro usuário.|Could|
|||US19 - Eu, como usuário, gostaria de ter acesso ao link para a videochamada em que eu terei minha aula.|Should|
|||US20 - Eu, como usuário, gostaria de poder avaliar outro usuário de uma nota de 0 a 10 para que ele possa ranqueado.|Would|
|||US21 - Eu, como usuário, gostaria de denunciar outro usuário, caso tenha acontecido algo grave comigo durante a aula.|Would|
||FE07 - Pesquisa e Filtro|US22 - Eu, como usuário, gostaria de poder buscar matérias para ver os professores disponíveis.|Should|
|||US23 - Eu, como usuário, gostaria de poder buscar professores em determinada matéria para que eu possa ver qual que me atende melhor.|Should|
|||US24 - Eu, como usuário, gostaria de filtrar professor por nível de escolaridade do conteúdo em determinada matéria para que eu possa ver a listagem dos que eu estou procurando.|Could|
|EP03 - Comunicação e Controle|FE08 - Contato|US25 - Eu, como usuário, gostaria de poder fazer perguntas para a administração da plataforma.|Would|
|||US26 - Eu, como usuário, gostaria de poder enviar depoimentos para a administração da plataforma.|Would|
|||US27 - Eu, como usuário, gostaria de poder entrar em contato com o administrador da plataforma para que eu possa me comunicar diretamente com alguém responsável pela plataforma.|Would|
||FE09 - Administração|US28 - Eu, como administrador, gostaria de poder ter acesso aos cadastros recentes de professores para que eu possa decidir aprovar ou não.|Must|
|||US29 - Eu, como administrador, gostaria de poder ter acesso às perguntas feitas pelos usuário para que eu possa respondê-las.|Could|
|||US30 - Eu, como administrador, gostaria de poder decidir criar “Pergunta Frequente” para ser disponibilizado para o público caso eu deseje.|Could|
|||US31 - Eu, como administrador, gostaria de poder visualizar as denúncias feitas pelos usuários para que eu possa entrar em contato com eles.|Would|
|||US32 - Eu, como administrador, gostaria de poder enviar e-mails para os usuários como forma de notificação e contato.|Would|
|||US33 - Eu, como administrador, gostaria de poder ver os depoimentos dos usuários enviados para a plataforma, e decidir se eu devo disponibilizá-los para o público.|Would|
|||US34 - Eu, como administrador, gostaria de poder criar códigos de desconto para que eu possa disponibilizar para os meus usuários.|Would|
|||US35 - Eu, como administrador, gostaria de poder criar novas matérias dentro da plataforma para que os professores possam se categorizar melhor.|Must|
|||US36 - Eu, como administrador, gostaria de poder gerenciar os usuários para que eu possa tomar as minhas próprias decisões em relações à eles. |Must|
